## Mise en ligne GitHub Pages
1. Crée ou ouvre ton dépôt GitHub.
2. Dépose tous les fichiers à la racine.
3. Va dans Settings > Pages.
4. Choisis Deploy from a branch.
5. Sélectionne main et /root.
6. Enregistre.
